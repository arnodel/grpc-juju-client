DeployRequest

Request message for the Deploy RPC.

FieldTypeLabelDescription
artifact_name string

Charm or bundle to deploy.

application_name string

Name of application to deploy (optional, if unspecified it will be taken from the artifact name).

channel string

Channel to use when deploying a charm or bundle from the charm store, or charm hub (optional).

series string

The series on which to deploy (optional).

dry_run bool

If true, just show what the bundle deploy would do.

RemoveRequest

Request message for the Remove RPC.

FieldTypeLabelDescription
application_name string

Name of application to remove

force bool

If true, completely remove the application and all its dependencies

no_wait bool

If true, rush through application removal without waiting for each individual step to complete
